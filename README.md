# 잡아봐유 (JABABWAYU)

**"도심 속에서 펼쳐지는  추격전!"**

**잡아봐유**는 어릴 적의 경찰과 도둑 놀이를 모바일 기술로 재해석한 **지역 기반 안드로이드 앱**입니다.
사용자 근처의 경찰과 도둑 모임을 만들고, 다른 사람이 만든 모임에 참여할 수 있습니다.

---

## 🛠 사용된 기술 (Tech Stack)

* **Language**: Kotlin
* **UI Toolkit**: Jetpack Compose
* **DB**: Firebase
* **Authentication**: Google login
* **Map / GPS**: Kakao Maps API
* **QR**: ZXing Library

* **개발 기간**: 2026.01.08 ~ 2026.01.14

---

## ✨ 주요 기능 (Key Features)

### 1. 🏠 홈
* **Google 로그인**: Google 로그인 후 닉네임, 태어난 연도 설정.
* **아바타 커스터마이징**: 레벨에 따라 다양한 아바타 및 복장 해금.
* **전적**: 지금까지의 모임 전적 확인.
* **신용도**: 비매너 유저를 필터링하기 위한 신뢰도 시스템.
* **게임 규칙**: 홈 화면에서 게임 규칙 확인.

### 2. 🗺️ 지도
* **Kakao Map 연동**: 모임을 시각적으로 확인.
* **모임 만들기 및 참가하기**: 원하는 조건의 모임 생성 및 참가.
* **스마트 알림**: 모임 시작 24시간/1시간 전 알림.
* **필터링**: 모임 리스트에서 지역별, 신용도 기준별, 뒤풀이 여부별 검색.
* **정렬**: 다양한 조건으로 모임 정렬.

### 3. 👋 출석
* **QR 출석**: 방장이 생성한 QR 코드를 참여자가 스캔하여 출석.
* **게임 시작**: 출석 인원 확인 후 게임 시작.
* **역할 분배**: 경찰, 도둑 비율 결정 후 랜덤 역할 배정.
* **승패 판정**: 라운드가 끝나면 방장이 승패를 결정.

### 4. 👯 내 모임
* **모임 확인**: 속해 있는 모임 확인.
* **실시간 채팅**: 모임 참여자들끼리 채팅.

### 5. 🤵 마이페이지
* **알림 설정**: 모임 시작 전 알림 수신 여부 설정.
* **고객 문의**: 개발자에게 문의 작성.

## 💾 설치 및 실행 방법 (How to install)
* **최소 사양**: Android 8.0 (Oreo) API 26 이상
* **권장 사양**: Android 12.0 (Snow Cone) 이상
* **Android Studio로 실행**
    1. 프로젝트 클론
    ```bash
    git clone https://github.com/secret-joojoo/JABABWAYU-police_and_thief.git
    ```
    2. `google-services.json` 설정
        1. Firebase Console에서 프로젝트를 생성합니다.
        2. 프로젝트 설정'에서 `google-services.json`을 다운로드 하여 `app/` 경로에 위치시킵니다.
    3. Kakao map Native key 설정
        1. Kakao developers(<https://developers.kakao.com/>)에서 앱을 생성합니다.
        2. '플랫폼 키'를 클릭합니다.
        3. 네이티브 앱 키를 클릭합니다.
        4. 패키지명과 키 해시를 입력하고 저장합니다.
        5. 네이티브 키를 복사하여 `local.properties` 파일에
           ```bash
           kakao.api.key={YOUR_NATIVE_APP_KEY}
           ```
           를 입력합니다. 이때, {YOUR_NATIVE_APP_KEY}에는 따옴표 없이 없이 키 값만 입력합니다.
    4. 빌드 및 실행
        1. Android Studio에서 `Sync Project with Gradle Files`를 클릭합니다.
        2. Android 12.0 (Snow Cone) 이상의 안드로이드 기기를 디버깅 모드로 연결 후 실행합니다 (Run 'app').

* **APK 다운로드**: [잡아봐유 다운로드 하기 (Google Drive)](https://drive.google.com/file/d/1kURJDZKWvdwkvjdyDe-lrSZevrc6KqZc/view?usp=drive_link)
